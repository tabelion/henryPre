# Homework: Javascript IV

## Instrucciones
---
1. En un archivo de texto separado que debes crear, escribe explicaciones de los siguientes conceptos como si se lo estuvieras explicando a un niño de 12 años. Hacer esto te ayudará a descubrir rápidamente cualquier agujero en tu comprensión.

	* Objetos: es una coleccion  de datos relacionados (variables y funciones que se denominan propiedades y metodos)
	* Propiedades: es una asociación entre un nombre (o clave) y un valor.
	* Métodos: las propiedades pueden ser una funsion dentro de un objeto y se las denomina metodo
	* Bucle `for…in`: es la iteracion de datos repetitivo una cantidad x de veces.
	* Notación de puntos vs notación de corchetes es ula forma de mostrar una variable de objeto separada el objeto y la propiedad por punto o corchete

2. Desde la carpeta `Prep` en la carpeta donde clonaste el repo, ingresa el comando `npm test JSIV.test.js` para correr los tests automatizados. Al principio, todos los tests estarán fallados/rotos. Encontrarás las funciones para hacer pasar los tests en el archivo `homework.js`.
